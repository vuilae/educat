## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/vuilae/educat.git
   
2. After cloning, navigate into the project directory
3. Install dependencies::

   ```bash
   npm install
   
4. Run the Server:

   ```bash
   npm start
